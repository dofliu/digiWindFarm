# 智慧型風場監控與維護系統

一個為大型風力發電場設計的現代化即時監控儀表板，提供一目了然的總覽、詳細的風機診斷，以及整合了 AI 驅動分析的維護管理系統。

## 核心功能

-   **即時 SCADA 儀表板**:
    -   全面的風場總覽，顯示所有風機的狀態。
    -   每台風機的詳細視圖，使用互動式儀表和趨勢圖顯示關鍵效能指標（KPI），如發電功率、轉子速度和風速。
-   **AI 驅動的故障診斷**:
    -   與 Google Gemini API 整合，即時分析風機故障數據。
    -   為操作員提供可能的原因和可行的建議，以實現更快、更準確的維護派遣。
-   **整合式維護中心**:
    -   管理技師名冊及其值班狀態。
    -   根據 AI 分析結果，直接從風機詳細視圖中創建和派遣工單。
    -   追蹤工單進度，並允許技師新增備註和上傳現場照片以完成任務。
-   **響應式設計**: 一個簡潔的深色主題 UI，完全響應式，適用於桌面和行動裝置。

## 技術棧

-   **前端**: React 19, TypeScript
-   **樣式**: Tailwind CSS
-   **圖表**: Recharts
-   **AI 整合**: Google Gemini API (`@google/genai`)
-   **模組載入**: 使用 Import Maps 的原生 ES 模組

## 如何開始

此專案為靜態網頁應用，無需建置步驟。

1.  確認您使用現代化的網頁瀏覽器。
2.  在瀏覽器中開啟 `index.html` 檔案即可執行。
3.  若要啟用 AI 故障診斷功能，您必須在環境中設定 `API_KEY`。
